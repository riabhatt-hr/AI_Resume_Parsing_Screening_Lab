# AI-Based Resume Screening and Parsing Lab

##  Overview
This project demonstrates how AI-assisted resume screening works by simulating an Applicant Tracking System (ATS). It focuses on converting unstructured resume data into structured information and ranking candidates based on job relevance.

---

##  Objective
- Create a synthetic dataset of 20 candidate resumes  
- Simulate resume parsing using text extraction logic  
- Extract key information such as:
  - Skills  
  - Experience  
  - Education  
- Compare resumes with a job description  
- Perform ATS-style keyword matching  
- Rank candidates based on relevance  
- Identify best-fit candidates for HR review  

---

##  Dataset
- File: `resume_parsing_screening_output.xlsx`  
- Contains parsed candidate data and ranking results  

---

##  Tools & Technologies Used
- Microsoft Excel  
- Basic AI concepts (Resume Parsing, ATS filtering)  
- Data structuring and keyword matching techniques  

---

##  Key Features
- Conversion of unstructured resumes into structured data  
- Keyword-based filtering similar to ATS systems  
- Candidate comparison based on extracted attributes  
- Ranking system to identify top candidates  

---

##  Insights
- Demonstrates how AI simplifies resume screening  
- Highlights the role of keyword matching in candidate selection  
- Shows how structured data improves decision-making efficiency  

---

##  How to Use
1. Download the dataset (`resume_parsing_screening_output.xlsx`)  
2. Open in Excel  
3. Review extracted fields (skills, education, experience)  
4. Analyze candidate ranking and selection criteria  

---
